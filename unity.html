Simple 3D Platformer

C# Gameplay Systems Level Design

Overview

A vertical-slice platformer featuring dynamic level scripting, advanced lighting techniques with VFX graph, and custom 3D models. This project evolved from a semi-complete prototype into a polished final release with revised visuals, responsiveness, and progression.

Core Pillars

  • Layered movement system with wall running.
  • Dynamic lighting and camera effects for a more atmospheric environment.

Tech Highlights

  • Editor tooling for level sequencing and checkpoint control.
  • Centralized event bus for UI, audio, and gameplay hooks.
